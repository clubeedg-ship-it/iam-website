Frequently Asked Questions

How long does setup really take?
With the included quick start guide, anyone can set up the Mobile Floor within 10-15 minutes. The flight case is cleverly organized so all components are in the right order. The projection and sensors are simply mounted on the stand, cables connected, and the system starts automatically.
What's included in the flight case?
The flight case contains: 4K projector with mount, sensor system, height-adjustable stand, all necessary cables, mini-PC with software, remote control, manual, and spare parts. Everything is safely stored in foam and ready to use.
Can I use the Mobile Floor outdoors?
The Mobile Floor works best indoors. At covered outdoor locations without direct sunlight, use is possible, but expect reduced visibility. We always recommend an indoor location for optimal gaming experience.
Is the Mobile Floor the same as the fixed floor?
Yes! The hardware and software are identical to the fixed interactive floor. The only difference is the packaging: everything is in a wheeled flight case instead of a fixed installation. You get the same quality, same games, and same projection surface.
What happens if something breaks during rental?
For rental with support, all damage is covered by our insurance. For self-service rental, normal wear and tear is covered by basic insurance, but we recommend additional event insurance for full coverage. We often solve minor technical issues by phone.
